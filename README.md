# Argus_AI
The GitHub repository ArgusAI demonstrates a prototype implementation of an artificial intelligence system designed to detect emerging antibiotic supply disruptions through anomaly detection across simulated hospital usage data. The Python script uses an Isolation Forest model to identify deviations in antibiotic prescribing that could signal early shortages and visualizes alerts to support decision‑makers. Modular code structure, clear documentation, and visual outputs make the prototype a compelling proof‑of‑concept that can be expanded with real healthcare data, predictive epidemiological models, and integrated pharmaceutical supply chain monitoring to enhance operational readiness and medical resilience across Allied healthcare networks.
